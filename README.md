# eventpage
有品牌形象的行銷活動網頁，促進消費者登錄發票並且方便行銷人員彙整資料。

## Introduction
EventPage 是一個乾淨且回應靈敏的 Web 模板，旨在展示事件、活動或行銷活動。它易於定制，非常適合需要在線展示且設計時尚的活動。

## Features
- RWD版面設計
- 方便行銷人員將使用者上傳到console.log的資料打包下載
- 互動元素
